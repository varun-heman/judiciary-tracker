# India Judiciary Tracker

India Judiciary Tracker is a personal project by Varun Hemachandran. I put the first version together during the week of May 12, 2026, using Codex and Claude Code to help scrape public data, structure it, and build the interface.

The site tracks publicly available information about courts, sitting judges, court administration, selected ministry officials, transfer notifications, and related declarations. I update it from time to time as sources change or as I improve the data model.

All source data is scraped from official/public sources, but both AI and human error are possible across the site. Please do not treat anything here as definitive. If you quote or rely on this tracker, you do so at your own risk and should verify the underlying official source first.

The public data, code, and interface work are available on [GitHub](https://github.com/varun-heman/judiciary-tracker) under the Apache 2.0 license. You are welcome to inspect the repository, reuse the work under that license, or point out corrections.

I am not a developer by training, but I work closely with courts across India. This project is a small attempt to make scattered public information easier to navigate.
